# elf5_covid_19
This repository contains essential scripts used to run proteome-wide colocalisation screens as presented in Pietzner, Chua, et al. 2021 MedRxiv (https://www.medrxiv.org/content/10.1101/2022.01.17.22269283v1) DOI: https://doi.org/10.1101/2022.01.17.22269283

All scripts contained in this repository have been created to run on the HPC facility in Cambridge UK, and are not expected to run out of the box.
